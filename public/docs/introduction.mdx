# OpenGL 소개

이 튜토리얼을 찾아오신 것을 보니, 아마도 컴퓨터 그래픽스의 내부 동작을 이해하고 멋진 것들을 직접 만들어보고 싶으신가 봅니다. 직접 구현해보는 것은 매우 재미있고 유익하며, 그래픽스 프로그래밍에 대한 깊은 이해를 제공합니다. 하지만 이 여정을 시작하기 전에 고려해야 할 몇 가지 사항들이 있습니다.

## 사전 지식

OpenGL은 독립적인 플랫폼이 아닌 그래픽스 API이기 때문에, 특정 프로그래밍 언어가 필요하며 여기서는 C++를 사용합니다. 따라서 이 튜토리얼을 따라하기 위해서는 C++ 프로그래밍 언어에 대한 기본적인 이해가 필요합니다. 필요한 경우 고급 C++ 주제들도 설명할 예정이지만, C++의 전문가가 될 필요는 없습니다. 단, 'Hello World' 프로그램 이상을 작성할 수 있어야 합니다. C++ 경험이 부족하다면  <LinkBlock url="https://www.learncpp.com" text="Learn Cpp"/> 의 무료 튜토리얼을 추천드립니다.

또한, 과정 중에 수학(선형 대수학, 기하학, 삼각법)을 사용할 것이며, 필요한 수학적 개념들을 설명할 예정입니다. 하지만 제가 수학자가 아니기 때문에, 설명이 이해하기 쉽더라도 완벽하지 않을 수 있습니다. 따라서 필요한 경우 더 자세한 설명을 제공하는 좋은 자료들을 안내해드리겠습니다. OpenGL 학습을 시작하기 전에 수학적 지식에 대해 두려워하지 마세요. 대부분의 개념들은 기초적인 수학 지식으로도 이해할 수 있으며, 가능한 한 수학을 최소화하려고 노력할 것입니다. 대부분의 기능은 수학을 완벽히 이해하지 않아도, 어떻게 사용하는지만 알면 됩니다.

## 튜토리얼 구조

LearnOpenGL은 여러 개의 일반 섹션으로 나뉘어 있습니다. 각 섹션은 서로 다른 개념들을 자세히 설명하는 여러 챕터들을 포함하고 있습니다. 각 챕터는 왼쪽 메뉴에서 찾을 수 있습니다. 개념들은 선형적으로 가르쳐지며(특별한 지시가 없는 한 위에서 아래로 순서대로 학습하는 것을 추천합니다), 각 챕터는 이론적 배경과 실제 적용 방법을 설명합니다.

개념을 더 쉽게 이해할 수 있도록, 그리고 구조를 더 명확하게 하기 위해, 이 튜토리얼에는 박스, 코드 블록, 색상 힌트, 함수 참조가 포함되어 있습니다.

### 코드
웹사이트에서 다음과 같이 구문 강조가 적용된 어두운 회색 박스에 있는 작은 코드 조각들을 많이 찾을 수 있습니다:


```cpp

#include <iostream>
#include <string>

int main() {
    std::string message = "Hello, OpenGL!";
    std::cout << message << std::endl;
    return 0;
}
```

이러한 코드 조각은 전체 코드의 일부분만 보여주기 때문에, 필요한 경우 해당 주제에 필요한 전체 소스 코드에 대한 링크를 제공할 것입니다.

### 색상 힌트
일부 단어들은 특별한 의미를 가지고 있음을 명확히 하기 위해 다른 색상으로 표시됩니다. 예를 들어, <Green>glCreateShader</Green>는 OpenGL의 셰이더 생성 함수이고, <Blue>GL_VERTEX_SHADER</Blue>는 버텍스 셰이더를 나타내는 상수이며, <Red>main()</Red>은 프로그램의 진입점 함수입니다.

<Green>정의</Green> : 초록색 단어는 정의를 나타냅니다. 즉, 자주 듣게 될 중요한 측면/이름을 의미합니다.

<Red>프로그램 구조</Red> : 빨간색 단어는 함수 이름이나 클래스 이름을 나타냅니다. 

<Blue>변수</Blue> : 파란색 단어는 모든 OpenGL 상수를 포함한 변수를 나타냅니다.
